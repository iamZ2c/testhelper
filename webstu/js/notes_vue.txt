computed 和 method 区别在于 computed多次使用 只调用一次函数,有缓存效率更高.

类似于python 方法
computed:{
            countAge: function () {
                let count = 0
                for (let movie of this.movies){
                    count += movie.age
                }
            return count
            }
        },
